Traffic Signal Scheduler using OS Algorithms (SJF, Priority, Round Robin)
How to Use the Traffic Signal Scheduler
Overview

This system simulates traffic signal scheduling using different OS scheduling algorithms based on vehicle detection.

Scheduling Algorithms
  • Shortest Job First (SJF): Used when there are few vehicles. Prioritizes lanes with the least number of vehicles.
  • Priority Scheduling: Activates when emergency vehicles are detected. Prioritizes lanes with emergency vehicles.
  • Round Robin: Default algorithm for normal traffic conditions. Cycles through each lane in sequence.
How to Use
  1. Set vehicle counts either by uploading images or using manual input.
  2. The system will analyze the traffic conditions.
  3. Based on the vehicle counts, it will choose an appropriate scheduling algorithm.
  4. The simulation will show which lane gets the green signal.
  5. Use the Step button to advance the simulation one step at a time.
  6. Try the Auto button for continuous simulation.
  7. Reset the simulation at any time to start over.
